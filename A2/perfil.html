Projetos

Sistema de gestão interna

Desenvolvimento de um sistema fullstack completo para gestão de contratos, comunicação com clientes e operações internas de uma loja. – Estruturei a aplicação utilizando React.js no frontend e Node.js no backend, com PostgreSQL como banco de dados relacional para controle seguro de clientes, contratos e locações. – Desenvolvi templates HTML dinâmicos e templates de e-mails automatizados para diferentes localidades, otimizando a geração e envio de documentos personalizados. – Implementei estilização modular em CSS, componentes reutilizáveis e arquitetura escalável com Docker, separando responsabilidades entre backend (aluguel, migrations) e frontend (hooks, pages, utils). – Apliquei princípios de design responsivo e UX/UI, e desenvolvi recursos como geração de nota promissória, declaração de retirada, assinatura digital, protótipos de itens do acervo e identidade visual da loja. – Resultado: padronização da comunicação, aumento da produtividade e segurança de dados, além da redução significativa no tempo de criação de contratos. Tecnologias utilizadas: React.js | Node.js | PostgreSQL | Docker | HTML | CSS | JavaScript | Python (Django migrations).

Meclang

Plataforma educacional de idiomas, com navegação responsiva, foco em acessibilidade e UX. • Desenvolvido com React.js, estrutura modular com reuso de componentes • Layout responsivo com HTML/CSS e base em protótipos do Figma • Simulação de CRUD e consumo de dados com mock API • Estudo prático que está sendo refatorado com Vue.js para aprendizado da nova stack
